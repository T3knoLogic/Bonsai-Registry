# Base Ecosystem — Deep Dive

**Last updated:** March 2026  
**Scope:** Base L2 — DeFi, SocialFi, NFTs, infrastructure (Coinbase OP Stack)

---

## Overview

The Bonsai Registry includes a comprehensive Base ecosystem section covering Coinbase's Ethereum Layer 2 and its DeFi, SocialFi, and NFT verticals. Base is built on the OP Stack and launched in August 2023. It ranks among the top L2s by TVL (~$7B+), with Aerodrome leading DeFi, Friend.tech pioneering SocialFi, and Zora powering NFT creation.
